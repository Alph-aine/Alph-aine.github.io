About


The Maze is a 3D game that uses raycasting to render a 2D map into a 3D world!

I was inspired to create this project by the concept page I read on ALX intranet.

As an adventurous person, I am always looking for new challenges, and this project seemed like a perfect opportunity to learn new skills and push myself to the limit.


You can check the project on Github here

Also, meet me on twitter, linkedin and Github

Installation


$ git clone git@github.com:Alph-aine/The-Maze.git
make linux
./maze default.txt

Usage


Controls:

  • w: Move up
  • s: Move down
  • d: Move left
  • a: Move right

Contributing


Follow these steps to contribute to The Maze:

  • Read the source files in the 'src' folder and the header files in the 'inc' folder.
  • Clone the repo and make a new branch: `$ git checkout https://github.com/Alph-aine/The-Maze.git -b [name_of_new_branch].`
  • Add a feature, fix a bug, or refactor some code :)
  • Write/update tests for the changes you made, if necessary.
  • Update README.md if necessary.
  • Open a Pull Request with a comprehensive description of changes.
